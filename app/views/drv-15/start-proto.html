Search for property information from Land Registry

Find information about a property in England or Wales, even if you don’t own it. Search by address to find the owner, how far its general boundaries extend and whether it’s at risk of flooding.

You will need to order an official copy of the register if you need to prove property ownership, eg for a court case.

Start now

Before you start

Read guidance on getting information about property and land, including alternative sources of information.

You'll need the following to use the service:

  • a user account
  • a credit or debit card
  • a printer, if you want to print the summary to refer to later

You’ll need to search different registers if the property is in Scotland or Northern Ireland.

What you'll get

The summary can tell you

  • owner's name and address
  • what they paid for the property – you won’t get this if it was bought before April 2000, if no money changed hands, or if the owner didn’t buy it outright
  • if there is a mortgage on the property, the lender's name and address
  • if it's a leasehold property, how long the lease is for
  • the rough extent and location of the property

You'll be able to view the information on screen. You'll only be able to view it once. If you want to keep a copy, you'll need to print it out or take a screen shot.

You won't get

You'll need to use a different Land Registry service if you need more detailed information like

  • the full legal register entry
  • details of access rights, rights of way, and restrictions
  • to prove who owns a property, for example in a court case or for a visa application
  • the official title plan showing indicative boundaries
  • a copy of the deeds
  • details of previous owners
